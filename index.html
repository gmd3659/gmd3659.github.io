PokePricer - Pokemon TCG Card Prices Website

PokePricer was developed to enable the fast search for any card that the Pokemon Trading Card Game has released, and the recent prices of the desired card. Users are also able to create accounts and save their favorite cards for quick viewing later. This website was solo-developed, front-end and back-end, using Node.js, React.js, and MongoDB.

Other Projects

Hearthstone Card Finder - Website and Discord Bot

This project was developed as a Discord bot for Hearthstone players, and especially newer players who don't know all of the cards yet. The Discord bot could be added to any server and users could use the chat to search and look-up information about cards with different search commands and the information would be sent back by the bot in the chat. This application was made using Node.js and the website was made using Vue.js. Unfortunately the API used for the project is no longer functioning and the bot will forever be offline.

Take The Bate - Casual Game

Take The Bait was developed with a friend of mine as a project for when we were first learning to use MonoGame. The game involves casting a hook into the water and then contolling the hook with the mouse to collect fish while avoiding the garbage floating around in the water. As your hook goes deeper into the water, rarer fish spawn but the speed of the hook increases as well gradually increasing difficulty.

A* Maze Solving Algorithm

For this project we worked in a team of two to create a DLL in C++ that could solve mazes as fast as possible. The DLL would take in a 2D array of a maze with and utilize the A* search algorithm to quickly get from the start position to the desired end position.
